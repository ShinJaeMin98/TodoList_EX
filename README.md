# 📅 TodoList_React

**React.js** 학습을 목적으로 제작한 투두리스트입니다. <br>

## 📃 개요

- **React.js** 연습을 목적으로 간단하게 구현한 TodoList입니다.
- 아이디어 및 디자인은 '**리액트를 다루는 기술**'을 참고하였습니다.

## 💻 사용 기술

## 📜 간략한 설명

- 간단한 투두 리스트입니다.
- React의 **sweetalert2**와 **React-dom-confetti**를 사용하여 효과를 주었습니다.
